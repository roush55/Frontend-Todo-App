:التصميم

  • تصميم الشكل العام وتجهيز المهام وتسليمه
  • تحويل التصميم إلى HTML+ CSS
  • جعل التصميم متجاوب مع المتصفحات الصغيرة (Responsive)
  • جعل التصميم متوافق كحد أدنى مع IE9
  • استخدام SASS لجعل ألوان التصميم سهلة التغيير (Skin-able/Theme-able)

:تحسينات

  • إمكانية إخفاء القائمة الجانبية
  • إمكانية تعليم المهام كمنتهية
  • إمكانية إعادة ترتيب المهام والمجموعات
  • التوافقية مع أجهزة اللمس

:أخرى

  • تسليم المشروع النهائي في git repository مع ملاحظة استخدام git-flow
  • .إضافة لمساتك الخاصة في الأماكن التي تراها مناسبة مع ذكر سبب إضافتها
